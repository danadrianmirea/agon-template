Converts an integer to another number base as a string.

This will convert an integer to other number bases and return a string. Specify the base, for example '16' for hex, '2' for binary. No memory allocation is done, and no bounds checking is performed, so make sure the string you pass in is long enough.

Parameters
[in]valueThe value to convert.
[in]resultA pointer to a long enough string for the output.
[in]baseThe base to use
Returns
A pointer to result
