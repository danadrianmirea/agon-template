Converts an integer to another number base as a string. 

This will convert an integer to other number bases and return a string. Specify the base, for example \textquotesingle{}16\textquotesingle{} for hex, \textquotesingle{}2\textquotesingle{} for binary. No memory allocation is done, and no bounds checking is performed, so make sure the string you pass in is long enough.
