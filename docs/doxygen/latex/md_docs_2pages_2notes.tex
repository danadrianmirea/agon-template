\begin{DoxyItemize}
\item Ctrl-\/w s, ctrl-\/w v -\/ Split current window half horizontally or vertically with same file in it
\item Ctrl-\/w n -\/ Same thing, new buffer but horizontal
\item \+:vnew -\/ same thing, new buffer but vertical
\item Ctrl-\/w q -\/ Close
\item Ctrl-\/w arrows -\/ move between buffers
\item \+:vs filename -\/ open file in new vertical split

Use a venv to make the python scripts not pollute your own Python install with their junk.


\begin{DoxyEnumerate}
\item Run {\ttfamily python -\/m venv venv} from inside the main source directory
\item Activate the virtual env with {\ttfamily venv/bin/activate}
\item Install the dependences by typing {\ttfamily pip install -\/r tools/requirements.\+txt}
\item Deactivate it when you are done using {\ttfamily deactivate} 
\end{DoxyEnumerate}
