{\footnotesize\ttfamily void vdp\+\_\+clear\+\_\+buffer (\begin{DoxyParamCaption}\item[{uint16\+\_\+t}]{buffer\+\_\+id }\end{DoxyParamCaption})}



Clear the buffer with the given ID. 

If the buffer does not exist then this command will do nothing.

VDU 23, 0 \&A0, buffer\+Id; 2

Please note that this clears out all of the blocks sent to a buffer via command 0, not just the last one. i.\+e. if you have built up a buffer over time by sending multiple commands to write to the same buffer ID, this command will clear out all of those commands.

Calling this command with a buffer\+Id value of -\/1 (65535) will clear out all buffers.
