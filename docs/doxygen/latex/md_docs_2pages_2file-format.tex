 \hypertarget{md_docs_2pages_2file-format_autotoc_md15}{}\doxysection{\texorpdfstring{RGBA2222 format images}{RGBA2222 format images}}\label{md_docs_2pages_2file-format_autotoc_md15}
Save images inside the data/images directory

If you run {\ttfamily make images} the {\ttfamily png2rgb.\+py} tool will turn them into RGBA2222 format image data. This data contains the following

Header\+: 10 bytes
\begin{DoxyItemize}
\item NCOTB -\/ magic string meaning RGB2222 bitmap image
\item WIDTH -\/ 16 bit little endian
\item HEIGHT -\/ 16 bit little endian
\item BPP -\/ 1 byte
\end{DoxyItemize}

Pixel Data\+: Rest of file
\begin{DoxyItemize}
\item 1 byte per pixel, RGBA 2 bits each
\end{DoxyItemize}

To use this, read the first 10 bytes, check it starts with "{}\+NCOTB"{}. Then use the width \texorpdfstring{$\ast$}{*} height \texorpdfstring{$\ast$}{*} bpp to know the size of the image data in bytes.\hypertarget{md_docs_2pages_2file-format_autotoc_md16}{}\doxysection{\texorpdfstring{Fonts}{Fonts}}\label{md_docs_2pages_2file-format_autotoc_md16}
Fonts are made from image atlas files containing each character in ASCII order. Each character is turned into its own RGBA2222 format image, and a {\ttfamily .fon} definition file is created to go along with it.\hypertarget{md_docs_2pages_2file-format_autotoc_md17}{}\doxysection{\texorpdfstring{Tilemaps}{Tilemaps}}\label{md_docs_2pages_2file-format_autotoc_md17}
